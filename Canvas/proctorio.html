How to Setup Proctorio:

  • Install/use the Google Chrome browser. 
  • Install the Proctorio extension in Chrome (click to download Proctorio (Links to an external site.) , or click the Secure Exam Proctor on the navbar to your left)
  • Make sure you have a reliable WiFi connection and webcam. Not sure about your wifi? Click here to run a wifi test. (Links to an external site.)
  • Note: Proctorio has been vetted by CU's OIT department and is FERPA compliant (meaning your recording is secure and private), but should you wish to uninstall it after  each quiz/exam, simply right-click the Proctorio shield, select Manage Extensions, and then click Remove Extension.

Common Issues and Solutions

Access code required

  • Make sure you're in Chrome and your browser window isn't maximized
  • Completely quit and reopen Chrome
  • Make sure your version of Chrome is 80 or higher
  • Delete and (re)install Proctorio from  getproctorio.com
  • Clear Chrome's cache (click the three dots, select More Tools and Clear Browsing Data)
  • Make sure you're logged out of your Google account in Chrome  

NOTE: If you have Windows 10S, you will not be able to take the exam on your computer and will need to make other arrangements.

Houston, we have a problem (RAM or CPU warning)

  • This is merely a warning.  You should be able to take the quiz without any problems.  
  • Just make sure that all unnecessary programs are closed before you start the quiz.  
  • Click Start exam pre-checks and continue.

Can't install Proctorio or it's outdated

  • Make sure Chrome is not maximized.  
  • Two-finger click (or right-click) the Proctorio shield in the Chrome toolbar and select Remove from Chrome...
  • Reinstall Proctorio from  getproctorio.com .

Screen check fail

  • Make sure you clicked the screen that asks you to share your screen.  You have to share your screen, or you can't get in.  
  • If you don't see something like that, exit the quiz and click in again.   You should see it then.  
  • If it continues to fail, and you’re on a Mac:
    • Go into System Preferences and click Security & Privacy.  
    • Go to Screen Recording and make sure Google Chrome is checked.

Microphone/camera check fail

For Macs only

  • Go to the Apple menu and click System Preferences.  
  • Then click Sound and Input.  
  • From there, turn off ambient noise reduction, make sure the internal microphone is selected, and turn up the input volume.  
  • If that doesn't work,
    • go back to System Preferences
    • click on Security & Privacy and then Microphone (and/or Camera).  
    • Make sure Google Chrome is checked.  
    • If these options aren’t available or the above doesn’t work, check to make sure your version of MacOS is 10.13 or newer by going to the Apple menu and clicking About This Mac.

If the above process doesn't work, or you're on a PC:

  • Go into Settings on Chrome (click the triple dot menu).  
  • Under "Privacy and security," click Site settings.  
  • Click Camera and/or Microphone.  
  • Delete the exception for Canvas under the Block section (Allow is okay).

Picture check fail

If you're wearing glasses, remove them for the picture and/or change the lighting in the room.

No student ID

Any picture ID is fine, as long as it has your name and picture on it.

You get kicked out

  • Go back into the exam immediately.  There’s enough time built-in for a kick-out or two.  
  • If it keeps happening, contact your instructor or course coordinator immediately.

Stuck on "waiting for canvas.colorado.edu"

Refresh your browser page and reload the exam.

Can't log into Canvas

  • If you're on campus, make sure you're on UCB Wireless and not UCB Guest.  
  • If that doesn't work, go to  identikey.colorado.edu  and change your password (click  Forgot My Password ).

None of the above helped

If there's something else wrong, call tech support at 866-948-2039 or 480-428-0692.
